## Development server

Run `mvn spring-boot:run` for a dev server. Navigate to `http://localhost:8080/swagger-ui.html`.

## Running unit tests

Run `mvn test` to execute the unit tests via [Spring Boot Test](https://spring.io/guides/gs/testing-web).
